# Specification

## About

* This app is a markdown document editor.  
* Its function and design is simple, neutral and minimalistic.

## Structure

* The markdown is rendered in a central column. 1000px wide.  
* On either side of the central column is a margin. 25px wide each.  
* The left margin is coloured.  
* The right margin is invisible.

## Sections

* A section is a part of a markdown document.  
* It starts with a heading.  
* It ends just before the next header.  
* The very first section does not need to start with a heading.  
* There is always a token section at the end of the document.  
* If the markdown document is empty, only the token section will be displayed.  
* A section can be in a highlighted state.  
* Only one (or zero) section can be highlighted at a time.

## Behaviour

* When a section enters edit mode, two buttons appear in the right margin.  
* The top button is the save button.  
* The button just below is the exit edit mode button.  
* If an additional heading is added to the section, when it gets saved it turns into two sections.  
* If the heading is removed from a section, when it gets saved it gets merged with the previous section.

## Style

* Colours should generally be pastel colours.  
* The webpage background should be a very light grey.  
* The document background should be white.  
* The margin should be light grey.  
* Corners should be square. Do not round-off corners.

#### When a section is highlighted

* The margin should be light blue.  
* The markdown background should be slightly grey-white.

#### When a section is selected for editing

* The margin should be blue.  
* The markdown background should be more grey-white.  
* The font should be monospaced.  
* There is no scroll bar.  
* It horizontally expands to fit the content.  
* The save button should be green. It should just say "o" (&\#111;).  
* The exit edit mode button should be grey. It should just say "x" (&\#215;)  
* The buttons should be round and floating.

## Keyboard shortcuts

* The format of keyboard shortcuts: \<MODIFIER\> \+ \<COMMAND\>  
* The modifier on Windows/Linux: CTRL  
* The modifier on macOS: CMD  
* In the below, the modifier is represented by: MOD

#### State: No section is highlighted

* MOD \+ UP/DOWN : The centre-most section is highlighted.

#### State: A section is highlighted

* MOD \+ X : Ignore changes and exit the section edit mode.  
* MOD \+ O : Save changes and exit the section edit mode.

## Code

* Do not use any third party libraries.
